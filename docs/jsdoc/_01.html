Library: bidichecker.js
Overview
Testing utilities to check a web page for BiDi-related errors. This file has the top-level entry points for use as a Javascript API, along with JSON-based interface functions meant to be called from an external testing framework like WebDriver.

bidichecker.checkPage(shouldBeRtl, opt_element, opt_filters)
Checks the contents of the current web page, including all nested frames, for BiDi-related errors. If opt_element is specified, checks only within the given element of the DOM (including any nested frames). This is the main entry point for the Javascript API.
parameters
boolean shouldBeRtl Is the root expected to be right-to-left?
Element= opt_element The root element of the DOM subtree to be checked; if null, checks the entire current web page.
Array.= opt_filters Error suppression filters.
returns
!Array. Array of error objects for all failing checks.
bidichecker.checkPageToJson(shouldBeRtl, opt_element, opt_filtersJson)
JSON interface to checkPage.
parameters
boolean shouldBeRtl Is the element/page expected to be right-to-left?
Element= opt_element The root element of the DOM subtree to be checked; if null, checks the entire current web page.
string= opt_filtersJson Error suppression filters in a JSON string.
returns
string Array of bidichecker.Error objects for all failing checks, in JSON format. If no errors were found, returns an empty array ("[]").
